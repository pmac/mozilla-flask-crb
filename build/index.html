Welcome to the Open Web Device CRB

The Open Web Device Compliance Review Board is an independent, not-for-profit organization formed to define and evolve the process of encouraging API compatibility and competitive performance for devices running the B2G operating system.

Our Goals

  • Demonstrate broad industry interest in establishing compliance specifications
  • Agree on common compliance criteria
  • Leverage existing industry organizations for easier deployment of CRB
